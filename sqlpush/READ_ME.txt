0) GET FILES
Download the sqlstd folder on your local machine.

1) PYTHON REQUIREMENTS
Install the following packages:
pip install pandas
pip install pyodbc

2) SQL SERVER SET-UP
In SQL Server: 
- create a database called Flood_factor
- run the script CREATE_TABLE belgium_addresses

3) RUN PUSH
Run the exepush.py file in the terminal, specifying your local server. 
For example:
python exepush.py BE2354629W2
